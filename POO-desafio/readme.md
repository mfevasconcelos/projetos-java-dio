# Modelagem Diagramação de um Componente Iphone

Neste repositório apresento uma modelagem simples baseada no vídeo de apresentação de Steve Jobs no lançamento do Iphone (2007) representando a relação de hierarquia entre produtos que a empresa já obtinha e como, por meio de uma perfeita representação hierarquica, implementou as determinadas funções em um só dispositivo.
